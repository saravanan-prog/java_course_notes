# 🔢 1. Number Programs
### ✅ 1. Check Even or Odd

```
int num = 7;

if (num % 2 == 0) {
    System.out.println("Even");
} else {
    System.out.println("Odd");
}
```

### ✅ 2. Reverse a Number
```
int num = 1234;
int reverse = 0;

while (num != 0) {
    int digit = num % 10;
    reverse = reverse * 10 + digit;
    num /= 10;
}

System.out.println(reverse); // 4321
```
### ✅ 3. Palindrome Number
```
int num = 121;
int original = num, reverse = 0;

while (num != 0) {
    int digit = num % 10;
    reverse = reverse * 10 + digit;
    num /= 10;
}

if (original == reverse)
    System.out.println("Palindrome");
else
    System.out.println("Not Palindrome");
```
### ✅ 4. Armstrong Number (153)
```
int num = 153;
int sum = 0, temp = num;

while (temp != 0) {
    int digit = temp % 10;
    sum += digit * digit * digit;
    temp /= 10;
}

System.out.println(sum == num ? "Armstrong" : "Not Armstrong");

```
