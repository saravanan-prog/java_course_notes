# JAVA INTEFACE

   - Interface in Java is a special type of class that contains only method declarations (no implementation), and any class that uses it must implement those methods.


### 1. “Method declaration”

 Only writing method name, no code

   ``` 
     
     interface Animal {
        void sound();   // only declaration
     }



   ```

###  2. “Class must implement it”

  If a class uses interface → it must write the code
  
  ```
    class Dog implements Animal {
        public void sound() {
            System.out.println("Bark");
        }
    }


 ```

 ##  Example 1

 ```
      
    interface Animal {
        void sound();   // only declaration
    }

    class Dog implements Animal {
        public void sound() {
            System.out.println("Bark");
        }
    }

    public class Main {
        public static void main(String[] args) {
            Animal a = new Dog();
            a.sound();
        }
    }



 ```

 ## Example 2


 ```
    interface Animal {
        void sound();
    }

    class Dog implements Animal {
        public void sound() {
            System.out.println("Bark");
        }
    }

    class Cat implements Animal {
        public void sound() {
            System.out.println("Meow");
        }
    }

    public class Main {
        public static void main(String[] args) {

            Dog d = new Dog();
            d.sound();

            Cat c = new Cat();      // ✔ Simple ❌ Not flexible
            c.sound();

            Animal a;               //  (Best way ⭐)

            a = new Dog();
            a.sound();   // Bark

            a = new Cat();
            a.sound();   // Meow


        }
    }


 ```

 ## Abstract Class vs Interface

 | Feature              | Abstract Class      | Interface         |
| -------------------- | ------------------- | ----------------- |
| Methods              | With & without code | Only without code |
| Variables            | Can have variables  | Only constants    |
| Keyword              | `extends`           | `implements`      |
| Multiple inheritance | ❌ No                | ✅ Yes             |
| Object creation      | ❌ No                | ❌ No              |
