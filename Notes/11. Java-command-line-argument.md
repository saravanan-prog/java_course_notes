# 🖥️  Command Line Arguments

👉 These are inputs passed when running the program from terminal

🧩 Syntax
```
public static void main(String[] args)

👉 args is an array of strings

✅ Example
public class Main {
    public static void main(String[] args) {
        System.out.println(args[0]);
    }
}

▶️ Run from terminal:
java Main Hello
✅ Output:
Hello
```


# 🔁 Multiple Arguments
```
public class Main {
    public static void main(String[] args) {
        for (String arg : args) {
            System.out.println(arg);
        }
    }
}
▶️ Run:
java Main Java Python C++
Output:
Java
Python
C++
```


### 🔢 Convert to Numbers
```
int num = Integer.parseInt(args[0]);
System.out.println(num * 2);

⚠️ Common Errors

❌ No argument passed:
System.out.println(args[0]); // error

👉 Fix:

if (args.length > 0) {
    System.out.println(args[0]);
}

```



### 🚀 Real Example (Add Numbers)
```
public class Main {
    public static void main(String[] args) {
        int a = Integer.parseInt(args[0]);
        int b = Integer.parseInt(args[1]);

        System.out.println(a + b);
    }
}
Run:
java Main 5 10
Output:
15
```
