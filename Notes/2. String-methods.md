# Creating Strings
#### 1. Using String Literal (Recommended)
```
String s1 = "Hello";
```


#### 2. Using new Keyword
```
String s2 = new String("Hello");
```


## 🔹 Common String Methods

#### Length
```
String s1 = "Hello";
int length = s.length();

```

#### Convert to Upper/Lower Case
```
String s1 = "Hello";

String upperCase = s.toUpperCase();
String lowerCase = s.toLowerCase();
```

#### Check Equality
```
String s1 = "Hello";
String s2 = "World";

s1.equals(s2);        // compares values
s1 == s2;            // compares references
```


#### Concatenation
```
String s1 = "Hello";
String s2 = "World";

String result = s1 + s2;

```

#### Substring

```
String s = "Hello";
String substr = s.substring(0, 3);

```

#### Contains

```
String s = "Hello";
boolean isAvailable = s.contains("ell");

```
#### Replace

```
String s = "Hello";
s.replace("H", "J")

```
