# throws :

```
    👉 throws is used to tell Java (and the caller):
    “This method may produce an exception”

```

In simple word

```
    throws = “I am not handling the error, someone else should handle it”

```
###  Basic Syntax

```
    void methodName() throws ExceptionType {
        // code
    }
```

#### Example 1 (File Handling)

```
import java.io.FileReader;

public class Test {
    public static void main(String[] args) throws Exception {
        FileReader file = new FileReader("test.txt");
    }
}

👉 Instead of handling using try-catch, we use throws

```

#### Example 2 (Custom Method)

```
class Test {
    static void checkAge(int age) throws Exception {
        if (age < 18) {
            throw new Exception("Not allowed");
        }
    }

    public static void main(String[] args) {
        try {
            checkAge(15);
        } catch (Exception e) {
            System.out.println(e.getMessage());
        }
    }
}

```
