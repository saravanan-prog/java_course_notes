# 1️⃣ What is JDK?
#### The Java Development Kit is a toolkit used to develop and run Java programs.
##### It contains:
- Java Runtime Environment (JRE) – Runs Java programs
- Java Virtual Machine (JVM) – Executes bytecode
- javac compiler – Converts Java code into bytecode
# 2️⃣ Download JDK
- Open browser
- Go to the official website of Oracle Corporation
- Download the latest JDK version (JDK 17 or JDK 21)
#### Choose your operating system:
- Windows (.exe)
- macOS (.pkg)
- Linux (.tar.gz)
#### install the software
# 3️⃣ Set Environment Variables
#### Step 1 — Open Environment Variables
- Search Environment Variables in Windows
- Click Edit the system environment variables
- Open Environment Variables
#### Step 2 — Create JAVA_HOME
Add new variable:
```
Variable Name: JAVA_HOME
Variable Value: C:\Program Files\Java\jdk-21
```
#### Step 3 — Update PATH
- Edit Path and add:
```
%JAVA_HOME%\bin
```

# 4️⃣ Verify Installation
### Open Command Prompt and type:
```
  java -version
  javac -version
```
### Example output:
```
   javac 21
```
