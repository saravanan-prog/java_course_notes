# 1️⃣ Variables in Java
- A variable is a named memory location used to store data that can change during program execution. 🧠💾
### Example: 
- storing a number, 
- text, or 
- boolean value.
###  Syntax of Variable Declaration
```
dataType variableName = value;
```
Example:
```
int age = 25;

Here  

  int → data type
  age → variable name
  25 → value stored in memory

```

### Example Program
```
public class VariablesExample {
    public static void main(String[] args) {

        int age = 25;
        String name = "Saravanan";
        double salary = 50000.50;
        boolean isDeveloper = true;

        System.out.println(age);
        System.out.println(name);
        System.out.println(salary);
        System.out.println(isDeveloper);
    }
}
```
Output:
```
    25
    Saravanan
    50000.5
    true
```

# 2️⃣ Types of Variables in Java

### 1. Local Variables
- Declared inside a method.
##### Features:
- Created inside methods
- Accessible only within the method
```
public void display() {
    int number = 10;
}
```
### 2. Instance Variables
- Declared inside a class but outside methods.
##### Features:
- Belongs to object
- Each object has its own copy
```
class Student {
    int age;
}
```
### 3. Static Variables
- Declared using static keyword.
#### Features:
- Shared by all objects
- Stored once in memory

```
class Student {
    static String college = "ABC College";
}
```
### 4.  final Variable
- A final variable is a variable whose value cannot be changed once it is assigned.
- Syntax:
```
final dataType variableName = value;
```
Example:
```
final int MAX_VALUE = 100;
```

### Example Program
```
public class FinalExample {

    public static void main(String[] args) {

        final int age = 25;

        // age = 30;  ❌ Error

        System.out.println(age);
    }
}

```
