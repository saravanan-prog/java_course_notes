# 🔹 StringBuilder in Java

- StringBuilder is very similar to StringBuffer, 
- but with one key difference:
👉 It is NOT thread-safe, which makes it faster.

🔹 Why StringBuilder?

Like StringBuffer, it allows you to create mutable strings (you can change them without creating new objects).

🔹 Example

```
    StringBuilder sb = new StringBuilder("Hello");
    sb.append(" World");
    sb.insert(5, ",");
    sb.replace(0, 5, "Hi");
    System.out.println(sb);  // Hi, World
```

🔹 Common Methods (same as StringBuffer)
#### ✅ append()
```
sb.append(" Java");  // adds at end
```
#### ✅ insert()
```
sb.insert(2, "y");  // adds at index
```
#### ✅ replace()
```
sb.replace(0, 2, "Hi");
```
#### ✅ delete()
```
sb.delete(2, 5);
```
#### ✅ reverse()
```
sb.reverse();
```
