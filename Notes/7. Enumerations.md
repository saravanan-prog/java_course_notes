
# Java, Enumerations

 - In Java, Enumerations (enums) are a special data type used to define a fixed set of constants. 
 
 - They make your code more readable, type-safe, and maintainable compared to using plain constants like int or String.

 ```
 enum Day {
    MONDAY,
    TUESDAY,
    WEDNESDAY,
    THURSDAY,
    FRIDAY,
    SATURDAY,
    SUNDAY
}

```

You can use it like this:

```

    Day today = Day.MONDAY;

    if (today == Day.MONDAY) {
        System.out.println("Start of the work week!");
    }


```

#### 🔹 Why Use Enums?
- Prevent invalid values (type safety)
- Improve readability
- Group related constants
- Can have methods and fields (unlike constants)



🚀 Real Working Example (E-commerce Order Status)

#### 🔹 Step 1: Define Enum

```
enum OrderStatus {
    PENDING,
    SHIPPED,
    DELIVERED,
    CANCELLED
}
```

#### 🔹 Step 2: Use Enum in a Class

```
class Order {
    int orderId;
    OrderStatus status;

    Order(int orderId, OrderStatus status) {
        this.orderId = orderId;
        this.status = status;
    }

    void printStatus() {
        switch (status) {
            case PENDING:
                System.out.println("Order is still pending.");
                break;
            case SHIPPED:
                System.out.println("Order has been shipped.");
                break;
            case DELIVERED:
                System.out.println("Order delivered successfully.");
                break;
            case CANCELLED:
                System.out.println("Order was cancelled.");
                break;
        }
    }
}

```
#### 🔹 Step 3: Main Method (Run the Program)

```
public class Main {
    public static void main(String[] args) {
        Order order1 = new Order(101, OrderStatus.PENDING);
        Order order2 = new Order(102, OrderStatus.DELIVERED);

        order1.printStatus();
        order2.printStatus();
    }
}
```
