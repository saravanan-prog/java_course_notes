# Java Abstraction

- Abstraction in Java is one of the core concepts of Object-Oriented Programming (OOP). 

- It means hiding implementation details and showing only the essential features of an object.

### 🔹 Simple Definition

```
Abstract class = a class that is not complete

It can have:
    methods with code ✅
    methods without code (abstract) ✅
    
```

### 🔹 Real-Life Example

```
Think of a car 🚗

You use the steering, brakes, and accelerator
You don’t need to know how the engine works internally

👉 That’s abstraction!

```


##  How Abstraction is Achieved in Java

Java provides two main ways:

### 1. Abstract Class

 Declared using abstract keyword

Can have both:

```
Abstract methods (no body)
Concrete methods (with body)
```


```
abstract class Animal {
    abstract void sound();  // abstract method
    
    void sleep() {
        System.out.println("Sleeping...");
    }
}

```
#### 2. Interface
