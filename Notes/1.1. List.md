# List

👉 List = Ordered collection that allows duplicates

- Maintains order
- Allows duplicates
- Supports index (0,1,2…)

```
    List
        ├── ArrayList   → uses array 📦
        ├── LinkedList  → uses links 🔗
        └── Vector      → thread-safe (old)

```

#### Simple Idea

```
    Index:   0   1   2   3
    Data:   [A] [B] [C] [A]

```
#### Explanation (Step-by-step)
```
1. add()            → adds elements
2. get(index)       → access element
3. add(index,item)  → insert at position
4. remove(index)    → delete element
5. size()           → count elements
6. clear()         -  delete all list

```

#### 1. add()

👉 Add element

```
list.add("A");
[A]

```
#### 2. add(index, element)

👉 Insert at specific position
```
list.add(1, "X");

Before: [A, B, C]
After:  [A, X, B, C]

```

#### 3. get(index)

👉 Access element
```
list.get(0);
Output: A
```

####  4. set(index, element)

👉 Update element
```
list.set(1, "Z");
Before: [A, B, C]
After:  [A, Z, C]
```
#### 5. remove()

👉 Remove element
```
list.remove(1);     // by index
list.remove("A");   // by value
```
#### 6. size()

👉 Number of elements
```
list.size();
[A, B, C] → 3
```

#### 7. contains()

👉 Check element exists
```
list.contains("A");
```
#### 8. isEmpty()

👉 Check if list is empty
```
list.isEmpty();
```
#### 9. clear()

👉 Remove all elements
```
list.clear();

```
[]
#### 10. indexOf()

👉 Get index of element
```
list.indexOf("B");
Output: 1
```


# Java Program (ArrayList using List)

```
import java.util.*;

public class Main {
    public static void main(String[] args) {

        // Creating List
        List<String> list = new ArrayList<>();

        // Adding elements
        list.add("A");
        list.add("B");
        list.add("C");
        list.add("A"); // duplicate allowed

        // Printing list
        System.out.println("List: " + list);

        // Access element
        System.out.println("Element at index 1: " + list.get(1));

        // Insert element
        list.add(1, "X");
        System.out.println("After inserting X: " + list);

        // Remove element
        list.remove("C");
        System.out.println("After removing C: " + list);

        // Size
        System.out.println("Size: " + list.size());
    }
}
📌 Output
        List: [A, B, C, A]
        Element at index 1: B
        After inserting X: [A, X, B, C, A]
        After removing C: [A, X, B, A]
        Size: 4
```
