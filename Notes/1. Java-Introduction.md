## 1️⃣ What is Java?
#### Java is a high-level, object-oriented programming language used to develop:
-    Web applications
-    Enterprise software
-    Android applications
-    Desktop applications

#### Java was created by James Gosling at Sun Microsystems in 1995 (later acquired by Oracle Corporation).
#### Key Features of Java
- Platform independent (Write Once Run Anywhere)
- Object-Oriented Programming
- Secure
- Robust
- Multithreading support
- Large community support
#### Example Java Program
```
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}

```
# 2️⃣ What is JDK?
- The Java Development Kit is a software development kit used to develop Java applications.
- It provides the tools required to write, compile, and run Java programs.
- Without JDK, you cannot create Java programs
# 3️⃣ Components of JDK
###  1️⃣ Compiler
#### javac compiler
#### Purpose:
- Converts Java source code (.java) into bytecode (.class)

### 2️⃣ JVM
#### Java Virtual Machine
#### Purpose:
- Executes Java bytecode
- Converts bytecode into machine code

### 3️⃣ JRE
#### Java Runtime Environment
#### Purpose:

- Runs Java programs.

#### Contains:
- Libraries
- JVM

## Relationship Between Java, JDK, JRE, and JVM

```
Java Program
     ↓
JDK
 └── JRE
      └── JVM
```
####  Flow

```
Java Code (.java)
       ↓
javac (Compiler)
       ↓
Bytecode (.class)
       ↓
JVM
       ↓
Machine Code
       ↓
Program Output

```
