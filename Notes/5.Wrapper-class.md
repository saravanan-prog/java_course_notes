# Wrapper Class
-  A wrapper class is a class that “wraps” a primitive data type (or)
-  another object so it can be used as an object with additional functionality


### 📦 In simple terms

It converts a basic value into an object so you can:

- use methods on it
- store it in collections (like arrays, lists, etc.)
- treat everything uniformly as objects


### 🧠 Example in Java

Java has primitive types like:

```
    int, double, char, boolean
```


| Primitive | Wrapper Class |
| --------- | ------------- |
| int       | Integer       |
| double    | Double        |
| char      | Character     |
| boolean   | Boolean       |


Example:

```

    int num = 10;                          // primitive
    Integer obj = Integer.valueOf(num);   // wrapper object

```
Or simply:

```

Integer obj = 10;  // autoboxing

```
