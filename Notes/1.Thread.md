# 🧵 What is a Java Thread?

- A thread in Java is a lightweight process 
- that allows your program to do multiple tasks at the same time (multitasking).

##### 👉 Example:

- Playing music 🎵
- Downloading file 📥
- Typing code ⌨️
```
All happening together = multithreading
```
## ⚙️ Ways to Create a Thread
```
class MyThread extends Thread {
    public void run() {
        System.out.println("Thread running...");
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread t = new MyThread();
        t.start();
    }
}

🖥️ Output
    Thread running...
```

✅ 2. Using Runnable (Better 👍)

```
class MyRunnable implements Runnable {
    public void run() {
        System.out.println("Runnable thread...");
    }
}

public class Main {
    public static void main(String[] args) {
        Thread t = new Thread(new MyRunnable());
        t.start();
    }
}


🖥️ Output
Runnable thread...
```

### States:
```
New                 → created
Runnable            → ready
Running             → executing
Waiting / Blocked   → paused
Terminated          → finished
````
