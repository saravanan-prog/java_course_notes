# 🔁 Autoboxing & Unboxing in Java

These are automatic conversions between primitive types and their wrapper classes.

#### 🔹 1. Autoboxing (Primitive ➝ Object)

Definition:
- Automatically converting a primitive into its wrapper class object.

✅ Example:

```
int num = 10;
Integer obj = num;   // autoboxing

👉 Java internally does:

Integer obj = Integer.valueOf(num);

```

#### 🔹 2. Unboxing (Object ➝ Primitive)

Definition:

 - Automatically converting a wrapper object into a primitive.

✅ Example:
```
Integer obj = 20;
int num = obj;   // unboxing
```
👉 Java internally does:
```
int num = obj.intValue();


```
🔁 Full Example

```
public class Main {
    public static void main(String[] args) {
        int a = 5;

        // Autoboxing
        Integer obj = a;

        // Unboxing
        int b = obj;

        System.out.println(a);
        System.out.println(obj);
        System.out.println(b);
    }
}
```
