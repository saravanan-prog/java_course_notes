# 🔁 Java Loops

👉 Loops are used to repeat a block of code multiple times

🧩 Types of Loops in Java

    - for loop
    - while loop
    - do-while loop
    - Enhanced for (for-each)

# 🔵 1. for Loop
```
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

✅ Output:
1 2 3 4 5

🧠 Structure:
for (initialization; condition; update)

✅ Use:

When you know how many times to loop
```

# 🟢 2. while Loop
```
int i = 1;

while (i <= 5) {
    System.out.println(i);
    i++;
}

✅ Use:
When you don’t know exact iterations
```

# 🟡 3. do-while Loop
```
int i = 1;

do {
    System.out.println(i);
    i++;
} while (i <= 5);

🔥 Key Point:
👉 Runs at least once, even if condition is false
```

# 🔁 4. Enhanced for (for-each)
```
int[] arr = {1, 2, 3, 4};

for (int num : arr) {
    System.out.println(num);
}

✅ Use:
To loop through arrays or collections
Cleaner and shorter
```
⚙️ Loop Control Statements
# ⛔ break
```
for (int i = 1; i <= 5; i++) {
    if (i == 3) break;
    System.out.println(i);
}

👉 Stops loop completely
```


# 🔄 continue
```
for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);
}
👉 Skips current iteration
```



# 🔥 Nested Loop
```
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        System.out.print("* ");
    }
    System.out.println();
}
Output:
* * *
* * *
* * *
```
