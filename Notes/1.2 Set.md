# Set?

👉 Set = Collection that does NOT allow duplicates

- ✔ Unique elements only
- ❌ No duplicates

#### 📊 Simple Diagram
```
Add: A, B, C, A

Set:
[A] [B] [C]

```
👉 Duplicate A is ignored

#### 📌 Types of Set

```
Set
 ├── HashSet   → No order
 └── TreeSet   → Sorted

```
#### 📊 Behavior

```
🔹 HashSet
[A, C, B]  (random order)
🔹 TreeSet
[A, B, C]  (sorted)

```
#### Simple Flow

```
add()       → insert
remove()    → delete
contains()  → check
size()      → count
clear()     → empty

```




#### 📌 Java Program Example
```
import java.util.*;

public class Main {
    public static void main(String[] args) {

        Set<String> set = new HashSet<>();

        set.add("A");
        set.add("B");
        set.add("C");
        set.add("A"); // duplicate

        System.out.println("Set: " + set);
    }
}
📌 Output
        Set: [A, B, C]

```
