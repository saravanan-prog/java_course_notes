# 🧠 What is a Class?

A class is a blueprint (template).

### 👉 It defines:

- Variables (data)
- Methods (behavior)

✅ Example:

```
class Car {
    String color;
    int speed;

    void drive() {
        System.out.println("Car is driving");
    }
}

👉 Here:

- color, speed   →  properties
- drive()        →  behavior

```



# 🚗 What is an Object?

An object is a real instance of a class.

### 👉 Think:

- Class = Design of a car
- Object = Actual car 🚗

🔥 Creating Object

```
Car c1 = new Car();

👉 Breakdown:

        Car → class
        c1 → object name

```



new → creates object

### 🧪 Full Example

```
class Car {
    String color;
    int speed;

    void drive() {
        System.out.println("Car is driving");
    }
}

public class Main {
    public static void main(String[] args) {

        Car c1 = new Car();

        c1.color = "Red";
        c1.speed = 100;

        System.out.println(c1.color);
        System.out.println(c1.speed);

        c1.drive();
        
    }
}

📊 Output
Red
10
Car is driving

```

## 🧱 Memory Concept (IMPORTANT)

```
c1 ---> [Car Object in Heap]
         color = "Red"
         speed = 100

👉 Object stored in heap memory
👉 c1 is a reference variable
```

🔁 Multiple Objects

```
Car c1 = new Car();
Car c2 = new Car();

👉 Each object has its own data
```
