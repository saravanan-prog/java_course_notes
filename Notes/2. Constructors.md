# 🧠 What is a Constructor?

- A constructor is a special method used to initialize an object.
- 👉 It runs automatically when you create an object

## 🔑 Key Rules

- Same name as class ✅
- No return type ❌ (not even void)
- Called automatically when using new

## 🚗 Basic Example

```
class Car {
    String color;

    Car() {   // constructor
        System.out.println("Constructor called");
    }
}

public class Main {
    public static void main(String[] args) {
        Car c1 = new Car();
    }
}

📊 Output

Constructor called
```

## 🔹 Types of Constructors

#### 1️⃣ Default Constructor

```
class Car {
    Car() {
        System.out.println("Default constructor");
    }
}

👉 No parameters

```



#### 2️⃣ Parameterized Constructor

```
class Car {
    String color;

    Car(String c) {
        color = c;
    }
}

public class Main {
    public static void main(String[] args) {
        Car c1 = new Car("Red");
        System.out.println(c1.color);
    }
}

📊 Output

Red
```

🔥 Using this keyword
```
class Car {

    String color;

    Car(String color) {
        this.color = color;
    }
}
```

👉 this.color → instance variable
👉 color → parameter

## 🔁 Constructor Overloading

#### Multiple constructors in one class:

```
class Car {
    String color;
    int speed;

    Car() {
        color = "Black";
    }

    Car(String color) {
        this.color = color;
    }

    Car(String color, int speed) {
        this.color = color;
        this.speed = speed;
    }
}
```
# 🧱 How it works (Memory idea)
new Car("Red")

- → Constructor runs
- → Object created in heap
- → Values assigned

## ⚠️ Important Points

- If you don’t write constructor → Java gives default constructor

- If you write one → default is NOT created

- Constructors cannot be inherited


## 🧾 Final Summary

- Constructor initializes object
- Runs automatically
- Supports overloading
- Used with this keyword
