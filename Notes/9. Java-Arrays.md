# 📦 Java Arrays

👉 An array is a collection of elements of the same data type stored in a single variable.

### 🧩 1. Declaring an Array
```
int[] arr;
```

### 🧩 2. Creating an Array
```
arr = new int[5];
```

👉 Creates an array of size 5 (default values = 0)

### 🧩 3. Declaration + Initialization

```
  int[] arr = {10, 20, 30, 40, 50};
```

### 🔍 4. Accessing Elements

```
    System.out.println(arr[0]); // 10
    System.out.println(arr[2]); // 30

👉 Index starts from 0
```

### 🔄 5. Loop Through Array

✅ Using for loop

``` 
    for (int i = 0; i < arr.length; i++) {
        System.out.println(arr[i]);
    }
```

✅ Using for-each

```
    for (int num : arr) {
        System.out.println(num);
    }
```

### 📏 6. Array Length
```        System.out.println(arr.length); 
```

# 🧱 7. Types of Arrays
🔹 1D Array
```
    int[] arr = {1, 2, 3};
```
🔹 2D Array (Matrix)
```
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6}
};

Access:
System.out.println(matrix[1][2]); // 6
```


# 🔄 8. Common Operations

✅ Find Sum

```
int sum = 0;

for (int num : arr) {
    sum += num;
}
```
✅ Find Maximum

```
int max = arr[0];

for (int num : arr) {
    if (num > max) {
        max = num;
    }
}
```

✅ Reverse Array
```
for (int i = arr.length - 1; i >= 0; i--) {
    System.out.println(arr[i]);
}
```


# ⚠️ Common Errors

❌ Array Index Out of Bounds

```
System.out.println(arr[5]); // error

👉 Always use:

    i < arr.length
```

🚀 Real Example
```       
    int[] arr = {2, 4, 6, 8};

        for (int num : arr) {
            System.out.println(num * 2);
        }
```
