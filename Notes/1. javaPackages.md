 # What is a Java Package?

- A package is basically a folder (directory) 
- that contains related classes.

👉 Example:

```
    package com.example.myapp;

    This means the class belongs to the package:
    com → example → myapp

```


🔹 Why Use Packages?
- 📂 Organize code (group related classes)
- 🔁 Avoid class name conflicts
- 🔒 Control access (public, private, protected, default)
- 📦 Reusable components
