

## ✅ 1. Exception Handling (try-catch)

```
    try    → risky code
    catch  → handles error
```
program continues safely


### Basic Example

```
public class Main {
    public static void main(String[] args) {
        try {

            int result = 10 / 0;                  // error
            System.out.println(result);

        } catch (ArithmeticException e) {

            System.out.println("Cannot divide by zero");

        }
    }
}
```


## 2. ✅ Multiple Catch Blocks

```
try {
    int arr[] = new int[5];
    arr[10] = 50;
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("Array index issue");
} catch (Exception e) {
    System.out.println("General error");
}

```

## 3. ✅ finally Block

- finally always executes (used for cleanup)

```
try {
    int a = 10 / 2;
} catch (Exception e) {
    System.out.println("Error");
} finally {
    System.out.println("Always runs");
}
```


## 🔹 5. throws

Used in method signature to declare exceptions that a method might throw.

```
public void readFile() throws IOException {
    FileReader file = new FileReader("test.txt");
}


This means the caller must handle or declare it.
```
