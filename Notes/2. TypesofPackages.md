# Types of Packages

#### 1. Built-in Packages

 Java provides many built-in packages, like:

```
    java.util      →  utilities (ArrayList, HashMap)
    java.lang      →  core classes (String, Math)
    java.io        →  input/output
```

👉 Example:

```
   import java.util.ArrayList;
```


#### 2. User-defined Packages

 You can create your own package.

Step 1: Declare package

```
  package mypackage;

```

Step 2: Create class

```
    package mypackage;

    public class Hello {
        public void show() {
            System.out.println("Hello from package!");
        }
    }
```


### Using a Package


##### Import entire package:

```
  import mypackage.*;

```


##### Import specific class:

```    
  import mypackage.Hello;
```

##### Without import (fully qualified name):

```
mypackage.Hello obj = new mypackage.Hello();

````
