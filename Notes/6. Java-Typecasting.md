# 🔄 What is Type Casting?

👉 Type casting means converting one data type into another

Example:
```
int a = 10;
double b = a; // int → double

```

# 🧩 Types of Type Casting

### 1️⃣ Implicit Casting (Widening)

👉 Done automatically by Java
👉 Small → Bigger type

📌 Order:
byte → short → int → long → float → double

✅ Example:

```
    int a = 10;
    double b = a;
```
System.out.println(b); // 10.0

👉 No data loss ✅

### 2️⃣ Explicit Casting (Narrowing)

- 👉 Done manually by programmer
- 👉 Bigger → Smaller type

⚠️ Example:

```
double a = 10.7;
int b = (int) a;

System.out.println(b); // 10

👉 Decimal part is lost ❌

```

### 🎯 Why Casting is Needed?

👉 When:

- Assigning different data types
- Performing calculations
- Reading user input

## 🧠 Quick Summary
```
   -----------------------------------------------
   Type	            Conversion	            Example
   ---------------------------------------------
    Implicit	     small → big	    int → double
    Explicit	     big → small	    double → int
```
 
 🔥 More Examples
#### ✅ 1. Division Problem
```
    int a = 5, b = 2;

    System.out.println(a / b);        // 2
    System.out.println((double)a / b); // 2.5
```

👉 Casting fixes integer division

#### ✅ 2. Char to Int
```
char ch = 'A';
int num = ch;

System.out.println(num); // 65
```
#### ✅ 3. Int to Char
```
int num = 66;
char ch = (char) num;

System.out.println(ch); // B
```
