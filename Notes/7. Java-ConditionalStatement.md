# 🔀 Conditional Statements (Decision Making)

👉 Used to execute different code based on conditions

🧩 1. if Statement
```
int age = 18;

if (age >= 18) {
    System.out.println("You can vote");
}
```
✅ Use:

When you have only one condition

# 🔁 2. if-else
```
int age = 16;

if (age >= 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}
```
✅ Use:

When you have two choices

# 🔀 3. if-else if-else
```
int marks = 75;

if (marks >= 90) {
    System.out.println("A");
} else if (marks >= 75) {
    System.out.println("B");
} else if (marks >= 50) {
    System.out.println("C");
} else {
    System.out.println("Fail");
}
```
✅ Use:

Multiple conditions

# 🔄 4. Nested if
```
int age = 20;
boolean hasID = true;

if (age >= 18) {
    if (hasID) {
        System.out.println("Allowed");
    }
}
```
✅ Use:

Condition inside another condition

# ⚡ 5. Ternary Operator (? :)

👉 Short form of if-else

```
int age = 18;

String result = (age >= 18) ? "Adult" : "Minor";
System.out.println(result);
```

# 🔘 6. switch Statement

```
int day = 3;

switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    case 3:
        System.out.println("Wednesday");
        break;
    default:
        System.out.println("Invalid");
}
```

# 🔥 Modern Switch (Java 14+)
```
int day = 2;

String result = switch (day) {
    case 1 -> "Monday";
    case 2 -> "Tuesday";
    default -> "Invalid";
};
```

# 🧠 Comparison

| Statement    | Use Case            |
| ------------ | ------------------- |
| `if`         | Single condition    |
| `if-else`    | Two options         |
| `if-else if` | Multiple conditions |
| `switch`     | Fixed values        |
| `ternary`    | Short if-else       |
