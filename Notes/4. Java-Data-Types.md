# Data Types in Java
- A data type defines what type of value a variable can store and how much memory it uses. 💾
- Example:
```
    int age = 25;

    Here:

        int  → data type
        age  → variable
        25   → value

```
### Types of Data Types in Java
- Java has two main categories of data types.
```
Data Types
   |
   |--- Primitive Data Types
   |
   |--- Non-Primitive Data Types
```
#### 1️⃣ Primitive Data Types
- Primitive data types store simple values directly in memory.

| Data Type | Size    | Example |
| --------- | ------- | ------- |
| byte      | 1 byte  | 100     |
| short     | 2 bytes | 2000    |
| int       | 4 bytes | 50000   |
| long      | 8 bytes | 100000  |
| float     | 4 bytes | 10.5f   |
| double    | 8 bytes | 20.55   |
| char      | 2 bytes | 'A'     |
| boolean   | 1 bit   | true    |

Example
```
public class DataTypesExample {
    public static void main(String[] args) {

        int age = 25;
        double salary = 45000.75;
        char grade = 'A';
        boolean isJavaFun = true;

        System.out.println(age);
        System.out.println(salary);
        System.out.println(grade);
        System.out.println(isJavaFun);
    }
}
```
#### 2️⃣ Non-Primitive Data Types
- Non-primitive data types store references to objects instead of the actual value.

Examples:

- String
- Arrays
- Classes
- Objects
- Interfaces

```
String name = "Saravanan";

int[] numbers = {10, 20, 30};

class Example {
    int a = 100;
    int b = 60;
}

```
