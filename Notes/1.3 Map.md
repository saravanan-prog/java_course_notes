# What is Map?

- Map = Key → Value pairs

```
       Map
        ├── HashMap        → No order, fast ⚡ 
        ├── LinkedHashMap  → Maintains insertion order 📌
        └── TreeMap        → Sorted by keys 🔼
```



```
1 → A
2 → B
```
- ✔ Key is unique
- ✔ Value can repeat

# Common Map Functions

#### 1. put()

👉 Add key-value pair
```
map.put(1, "A");
Before: { }
After:  {1=A}
```

####  2. get()

👉 Get value using key
```
map.get(1);
Output: A
```
####  3. remove()

👉 Remove by key
```
map.remove(1);
Before: {1=A, 2=B}
After:  {2=B}
```
####  4. containsKey()

👉 Check if key exists
```
map.containsKey(1);
true / false
```
#### 5. containsValue()

👉 Check if value exists
```
map.containsValue("A");
```
#### 6. size()

👉 Number of key-value pairs
```
map.size();
```
#### 7. isEmpty()

👉 Check if map is empty
```
map.isEmpty();
```

#### 8. clear()

👉 Remove all entries
```
map.clear();
```

#### 🔹 9. keySet()

👉 Get all keys
```
map.keySet();
{1, 2, 3}
```

#### 10. values()

👉 Get all values
```
map.values();
[A, B, C]
```
#### 11. entrySet()

👉 Get key-value pairs
```
map.entrySet();
[1=A, 2=B]
```

# Java Program
```

import java.util.*;

public class Main {
    public static void main(String[] args) {

        // Create Map
        Map<Integer, String> map = new HashMap<>();

        // Add elements (key → value)
        map.put(1, "A");
        map.put(2, "B");
        map.put(3, "C");
        map.put(1, "D"); // duplicate key (will replace A)

        // Print map
        System.out.println("Map: " + map);

        // Get value using key
        System.out.println("Value of key 2: " + map.get(2));

        // Check key
        System.out.println("Contains key 3: " + map.containsKey(3));

        // Remove element
        map.remove(2);
        System.out.println("After removing key 2: " + map);

        // Print all keys
        System.out.println("Keys: " + map.keySet());

        // Print all values
        System.out.println("Values: " + map.values());

        // Size
        System.out.println("Size: " + map.size());
    }
}


📌 Output
            Map: {1=D, 2=B, 3=C}
            Value of key 2: B
            Contains key 3: true
            After removing key 2: {1=D, 3=C}
            Keys: [1, 3]
            Values: [D, C]
            Size: 2

```
