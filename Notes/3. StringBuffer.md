# StringBuffer

- StringBuffer is a class used in languages like Java to create mutable (changeable) strings.

- Normally, strings in Java are immutable (cannot be changed). Every time you modify a string, a new object is created.


- StringBuffer solves this by allowing you to modify the same object, which improves performance.

##### Key Features

- Mutable (can change content)
- Thread-safe (synchronized)
- Slower than StringBuilder due to thread safety
- Efficient for repeated string modifications


```
StringBuffer sb = new StringBuffer("Hello");

sb.append(" World");
sb.insert(5, ",");
sb.replace(0, 5, "Hi");

System.out.println(sb); 

```


## 🔹 Common Methods

```
    append()  →  adds text at the end
    insert()  →  adds text at a specific position
    replace() →  replaces part of the string
    delete()  →  removes characters
    reverse() →  reverses the string
```


#### 🔹 1. append() → adds text at the end
```
StringBuffer sb = new StringBuffer("Hello");
sb.append(" World");

System.out.println(sb);  // Hello World
```

#### 🔹 2. insert() → adds text at a specific position
```
StringBuffer sb = new StringBuffer("Hello World");
sb.insert(5, ",");

System.out.println(sb);  // Hello, World

👉 Index starts from 0

```

#### 🔹 3. replace() → replaces part of the string

```
StringBuffer sb = new StringBuffer("Hello World");
sb.replace(6, 11, "Java");

System.out.println(sb);  // Hello Java

👉 Replaces from index 6 to 10 (end index is exclusive)

```

#### 🔹 4. delete() → removes characters

```
StringBuffer sb = new StringBuffer("Hello World");
sb.delete(5, 11);

System.out.println(sb);  // Hello

👉 Deletes from index 5 to 10

```

#### 🔹 5. reverse() → reverses the string

```
StringBuffer sb = new StringBuffer("Hello");
sb.reverse();

System.out.println(sb);  // olleH

```
