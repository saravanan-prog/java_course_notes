# Java Access Specifiers

- Access specifiers define who can access a class, method, or variable.
- Java has 4 access modifiers:
```
    1. 🟢 public
    2. 🟡 protected
    3. 🔵 default (no keyword)
    4. 🔴 private
```


#### 1. 🟢 public

- Accessible from anywhere
- No restriction
```
public class Test {
    public int x = 10;
}
```
#### 2. 🟡 protected
- Accessible:
- Within the same package
- In other packages (only through inheritance)

```
protected void show() {
    System.out.println("Protected method");
}
```

#### 3. 🔵 default (no keyword)

- Accessible only within the same package
- If you don’t specify anything, it’s default

```
class Test {
    int x = 5; // default access
}
```

#### 4. 🔴 private

Accessible only within the same class
Most restrictive

```
private int secret = 100;

```

# Access Level Summary Table

| Modifier  | Same Class | Same Package | Subclass (Other Package) | Anywhere |
| --------- | ---------- | ------------ | ------------------------ | -------- |
| public    | ✅          | ✅            | ✅                        | ✅        |
| protected | ✅          | ✅            | ✅                        | ❌        |
| default   | ✅          | ✅            | ❌                        | ❌        |
| private   | ✅          | ❌            | ❌                        | ❌        |



# 🔹 Simple Way to Remember

🔓 public → everyone can access
🧬 protected → family (inheritance) + same package
📦 default → same package only
🔒 private → only inside the class
