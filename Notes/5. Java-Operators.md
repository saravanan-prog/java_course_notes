# Operators in Java
- An operator is a symbol used to perform operations on variables or values. ➕➖✖️➗
Example:
```
int a = 10;
int b = 5;
int sum = a + b;

Here + is an operator.
```

## Types of Operators in Java
- Java operators are divided into several categories.

```
Operators
   |
   |-- Arithmetic
   |-- Relational (Comparison)
   |-- Logical
   |-- Assignment
   |-- Unary
   |-- Bitwise
   |-- Ternary
```
## 1️⃣ Arithmetic Operators
 Used for mathematical calculations.

| Operator | Meaning             | Example |
| -------- | ------------------- | ------- |
| +        | Addition            | a + b   |
| -        | Subtraction         | a - b   |
| *        | Multiplication      | a * b   |
| /        | Division            | a / b   |
| %        | Modulus (Remainder) | a % b   |

Example

```
int a = 10;
int b = 5;

System.out.println(a + b); 
System.out.println(a - b);
System.out.println(a * b);
System.out.println(a / b);
System.out.println(a % b);

Output
15
5
50
2
0

```

| -------- | ------------------------ | ------- |
| ==       | Equal to                 | a == b  |

# ❓ 6. Ternary Operator

Shortcut for if-else

```
int age = 18;

String result = (age >= 18) ? "Adult" : "Minor";
System.out.println(result);
```

# 🔍 7. Bitwise Operators (Advanced)

Focus on these:
```
      - 👉 &  (AND) - "Both must be 1"      -> 1

               0101 (5)
            &  0011 (3)
             -----------
               0001 → 1

      - 👉 |  (OR)  - "At least one is 1"  -> 1
            

              0101
              0011
            -----------
              0111 → 7




      - 👉 ^  (XOR) - "Different = 1"

                 0101
               ^ 0011
               -----------
                 0110 → 6


```

Works at binary level

```
   int a = 5, b = 3;

   System.out.println(a & b); // AND
   System.out.println(a | b); // OR
   System.out.println(a ^ b); // XOR
```

### Check even/odd

 ```
   if ((num & 1) == 0)
      System.out.println("Even");

 ```
### Find unique number  
```
   int result = 0;
   int[] arr = {1, 2, 2, 1, 3};

   for (int num : arr) {
      result ^= num;
   }

   System.out.println(result); // 3

```
