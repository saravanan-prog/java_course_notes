# Operators in Java
- An operator is a symbol used to perform operations on variables or values. ➕➖✖️➗
Example:
```
int a = 10;
int b = 5;
int sum = a + b;

Here + is an operator.
```

## Types of Operators in Java
- Java operators are divided into several categories.

```
Operators
   |
   |-- Arithmetic
   |-- Relational (Comparison)
   |-- Logical
   |-- Assignment
   |-- Unary
   |-- Bitwise
   |-- Ternary
```
## 1️⃣ Arithmetic Operators
 Used for mathematical calculations.

| Operator | Meaning             | Example |
| -------- | ------------------- | ------- |
| +        | Addition            | a + b   |
| -        | Subtraction         | a - b   |
| *        | Multiplication      | a * b   |
| /        | Division            | a / b   |
| %        | Modulus (Remainder) | a % b   |

Example

```
int a = 10;
int b = 5;

System.out.println(a + b); 
System.out.println(a - b);
System.out.println(a * b);
System.out.println(a / b);
System.out.println(a % b);

Output
15
5
50
2
0

```

# 🔍 7. Bitwise Operators (Advanced)

Focus on these:
```
      - 👉 &  (AND) - "Both must be 1"      -> 1

               0101 (5)
            &  0011 (3)
             -----------
               0001 → 1

      - 👉 |  (OR)  - "At least one is 1"  -> 1
            

              0101
              0011
            -----------
              0111 → 7




      - 👉 ^  (XOR) - "Different = 1"

                 0101
               ^ 0011
               -----------
                 0110 → 6


```

Works at binary level

```
   int a = 5, b = 3;

   System.out.println(a & b); // AND
   System.out.println(a | b); // OR
   System.out.println(a ^ b); // XOR
```

# 8. Shift Operators?

- Shift operators move bits left or right.
- Think of a number in binary (0s and 1s), and we just shift those bits.


### 1️⃣ Left Shift <<
```
a << n

👉 Moves bits to the left by n positions
👉 Fills right side with 0
```

🔥 Example:
```
int a = 5;   // 00000101
int result = a << 1;
5  = 00000101
<<1
    00001010 = 10

✅ Result = 10

👉 Shortcut:

a << n = a * (2^n)
2️⃣ Right Shift >> (Signed)
a >> n
```
- 👉 Moves bits to the right
- 👉 Keeps the sign bit (important for negative numbers)

```
Example:
int a = 20;  // 00010100
int result = a >> 1;
20 = 00010100
>>1
     00001010 = 10

✅ Result = 10

👉 Shortcut:

a >> n = a / (2^n)
3️⃣ Unsigned Right Shift >>>
a >>> n
```

- 👉 Moves bits right
- 👉 Always fills left with 0 (ignores sign)

Example with negative number:
```
int a = -5;
System.out.println(a >> 1);
System.out.println(a >>> 1);

👉 Results will be different because:

>> keeps sign

>>> removes sign effect

⚡ Simple Visualization
Left Shift
5 << 2

00000101 → 00010100 = 20
Right Shift
20 >> 2

00010100 → 00000101 = 5
```
